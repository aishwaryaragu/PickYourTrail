## Setup
To run this project, install java locally:

```

$ set path C:\Program Files\Java\jdk1.8.0_60\bin
$ javac {FileName.java} --->For Compilation
$ java {FileName} --> For Run

OR

Use Java IDE (Eclipse)
$ save the java file in repository
$ run the program
